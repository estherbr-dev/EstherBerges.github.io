K.A.S.E.

Game Description

K.A.S.E. is a platformer game where you control a little robot that can interact and completely change the level layout. He can break down giant structures or move pieces around to create his own path. Thanks to this system the players can solve the levels in many different ways.

My responsiblities and missions

  • Programmed the physics and breakable structures system to allow more than 500 physical cubes on screen.
  • Designed and programmed the character and camera: created the player movements, player powers and camera behavior.
  • Implemented the game sounds and animations in Unity.
